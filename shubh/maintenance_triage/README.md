# 🔧 Maintenance Ticket Triage

**Ticket → Classify → Route to Team**

## What It Does
Classifies maintenance tickets by category (electrical, mechanical, software, etc.) and priority (critical/high/medium/low), then routes to the correct team. Auto-escalates safety risks.

## ROI
- **Manual time**: 5–10 min per ticket to read, classify, assign
- **Volume**: 10–50 tickets/day in mid-size plant
- **Savings**: 1–4 hours/day of supervisor time
- **Impact**: Faster response to critical issues, zero mis-routes

## Run Demo
```bash
python -m shubh.maintenance_triage.main
```
