# 🔬 QA Anomaly Detection

**Inspection Logs → Anomaly Flags + Executive Summary**

## What It Does
Analyzes CSV/Excel inspection logs using statistical methods (z-score, out-of-range, deviation). NO ML training — just smart statistics. LLM generates plain-English summary for plant managers.

## ROI
- **Manual time**: QA engineer reviews logs for 1–2 hrs/day
- **Missed anomalies**: Manual review catches ~60% of issues
- **Savings**: Earlier defect detection, reduced scrap rate, 2–5 hours/week

## Run Demo
```bash
python -m shubh.qa_anomaly.main
```
