# 📊 Production Report Auto-Generation

**Daily CSV → Executive Summary Report**

## What It Does
Reads daily production CSVs, aggregates output/downtime/quality/scrap metrics, then uses LLM to write a professional executive summary with highlights, concerns, and recommendations.

## ROI
- **Manual time**: 1–2 hours/day compiling daily reports
- **Quality**: Consistent formatting, never misses metrics
- **Savings**: 5–10 hours/week of operations manager time

## Run Demo
```bash
python -m shubh.production_report.main
```
