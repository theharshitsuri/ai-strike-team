# 📦 Inventory Restock Forecasting Agent

**Demand Forecast + Reorder Alerts**

## What It Does
Analyzes historical sales and current stock levels to forecast demand (moving average), calculate reorder points with safety stock, and generate restock recommendations. LLM writes a readable explanation for managers.

## ROI
- **Manual time**: 2–4 hours/week on inventory analysis
- **Stockout prevention**: Catches critical items before they run out
- **Overstock reduction**: Right-sized orders reduce carrying costs 5–10%

## Run Demo
```bash
python -m shubh.inventory_restock.main
```
